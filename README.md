# FitLoop

FitLoop is a modern activewear marketplace for discovering, buying, and selling sports clothing, sneakers, and accessories.

Users can browse new and pre-owned products, filter listings by category, size, brand, condition, and price, save favorite items, create their own listings, manage products from a dashboard, and complete a checkout flow.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- React Hook Form
- Zod
- Zustand
- TanStack Query

## Features

- Authentication
- Product creation and editing
- Catalog filters
- Search and sorting
- Favorites
- Cart
- Checkout flow
- User dashboard
- Responsive design
